Welcome to the Stock API

This API provides access to stock information.

Current data

  • GET /query/{ISIN} - get current data for given ISIN 2)

Historic data

  • GET /history - list ISIN and name of stocks in history db
  • GET /history/{ISIN} - latest record of given {ISIN} 2)
  • GET /history/{ISIN}/{COUNT} - latest {COUNT} records of given {ISIN} 2)
  • GET /daily/{ISIN} - latest daily price of given {ISIN} 2)
  • GET /daily/{ISIN}/{COUNT} - latest {COUNT} daily prices of given {ISIN} 2)

2) Using ISIN US5949181045 (Microsoft Corp.) as an example.
